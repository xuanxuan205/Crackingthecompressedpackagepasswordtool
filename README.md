## 📖 使用说明

### 基本操作
1. 打开程序并选择需要恢复的压缩包文件
2. 根据已知信息选择合适的模式
3. 设置密码长度、字符集或字典等参数
4. 点击开始并观察进度与结果

### 使用建议
- 优先尝试你最有把握的密码规律
- 尽量缩小密码长度范围，提高恢复效率
- 如遇到性能问题，可优先检查 GPU 驱动和系统资源占用

## 🧭 功能概览

- 支持 ZIP / RAR / 7Z 等常见压缩文件
- 支持字典、掩码、AI 辅助、混合、GPU 加速等模式
- 支持进度保存与继续恢复
- 支持 GPU 状态诊断与排障提示
- 支持更新检查、公告查看与在线教程入口
- 支持 QQ / 163 邮箱绑定与找回密码功能

---

## ⚙️ 系统要求

- Windows 10 / 11（64 位）
- 建议内存 4GB 及以上
- 建议预留 2GB 以上可用磁盘空间
- 如需 GPU 加速，建议使用支持 OpenCL 的 NVIDIA / AMD 显卡

---

## 🧪 GPU 与性能说明

- 程序支持检测 GPU 并显示运行状态
- 若出现兼容提示，通常仍可通过 OpenCL 方式使用
- 可导出 GPU 诊断信息，便于排查驱动、温度与利用率问题
- 不同压缩格式的校验开销不同，速度表现会有差异
